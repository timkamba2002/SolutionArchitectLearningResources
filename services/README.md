# AWS Services Learning Materials

This directory contains learning notes, key concepts, and practical examples for each AWS service covered in the Solution Architect certification.

Each service folder includes:
- **README.md** - Overview and key concepts
- **notes.md** - Detailed learning notes from videos
- **key-concepts.md** - Quick reference for important topics
- **practice.md** - Hands-on practice tasks and CLI commands
